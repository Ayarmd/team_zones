مبادئ رسم ERD — النسخة المصحّحة

مخطط علاقات الكيانات (ERD) يصف الكيانات والعلاقات بينها وفق نموذج Chen:

  • مستطيل = كيان (Entity) — جدول في قاعدة البيانات
  • معيّن ◇ = علاقة (Relationship) — فعل يربط كيانين
  • 1 / N / M = Cardinality — عدد التكرار على كل طرف
  • تخصص (Specialization) — المستخدم كيان واحد، لكن الدور يحدد علاقته

⚠️ قاعدة مهمة: لا تكتب «المستخدم يدير الصالة» — اكتب «المدير يدير» أو «موظف الصالة يعمل في».
