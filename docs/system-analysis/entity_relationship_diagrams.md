## شكل المخطط

مثل المثال الأكاديمي:

```
[الصالة] ——1—— ◇ تحتوي على ◇ ——N—— [الباقة]
```

- **مستطيل بنفسجي داكن** = كيان
- **معين بنفسجي فاتح** = علاقة (فعل)
- **1 / N / M** = Cardinality
